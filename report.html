Found a issue?

Please let us know at: bugreports@openvoid.net

Before sending an issue...

Please check the status page for an outage or maintance first.

When reporting a bug...

Be sure to give us as much information as possible like:

  • Steps to reproduce the bug.
  • Device used.
  • Web browser (if one is used).
  • Anything else to help fix the issue.

Please note that bugs may not be replied to if no further information is needed.

Only valid bug reports will be read from this email!
